# FBS Guard

Демонстрационный проект MoviBobby: интернет-магазин и операционная FBS-платформа.

## Возможности
- товарная витрина и демо-корзина;
- автоматический расчёт доставки по весу, габаритам, региону и перевозчику;
- расчёт маржи и рентабельности заказа;
- панель заказов, распределение остатков и мониторинг рисков;
- адаптивный интерфейс.

## Запуск
```bash
npm install
npm run start
```
